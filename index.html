Software Suite + Mobile App

Short Stack / RailBird

A full-stack poker room management platform paired with a companion mobile app for players. Built from scratch to replace paper waitlists and manual operations — now handling real-time table management, QR-code check-ins, tournaments, analytics, and a social network for players.

📱 QR Code Check-In

Players scan with the RailBird mobile app for instant check-in. Automatic player recognition and history tracking.

📋 Waitlist Management

Digital FIFO waitlists with automatic seat calling and support for multiple game types simultaneously.

🪑 Table Management

Visual seat assignments, drag-and-drop player management, and real-time status tracking across all tables.

🏆 Tournament System

Registration, customizable blind structures, automatic timers, and payout calculations.

📊 Analytics & Reporting

Player history, session tracking, game preferences, revenue reporting, and live dashboard statistics.

🔌 REST API

30+ RESTful API endpoints with complete documentation for integrations and custom extensions.

👥 Social Network

Players connect, share sessions, coordinate games, and receive push notifications through the RailBird app.

🔐 Enterprise Security

Role-based access control, SQL injection protection, and comprehensive audit trails.
